# GroupDocs.Search

Simple Angular app that allows to display and manipulate documents supported by groupdocs libraries APIs

# Getting started

Install using `npm`

`npm i @groupdocs.examples.angular/search`

or 

`yarn add @groupdocs.examples.angular/search`

Add following code within `<body>` tag

```JS
<gd-search></gd-search>
```

# Note
This component is intended to work with following GroupDocs examples:

- [GroupDocs Search for Java Spring](https://github.com/groupdocs-search/GroupDocs.Search-for-Java-Spring)
- [GroupDocs Search for Java Dropwizard](https://github.com/groupdocs-search/GroupDocs.Search-for-Java-Dropwizard)
- [GroupDocs Search for .NET WebForms](https://github.com/groupdocs-search/GroupDocs.Search-for-.NET-WebForms)
- [GroupDocs Search for .NET MVC](https://github.com/groupdocs-search/GroupDocs.Search-for-.NET-MVC)
